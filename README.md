# Cursor PRD Project

Репозиторий из MD-файлов для работы с Cursor: шаблоны PRD (Product Requirements Document) и инструкции для AI-агентов.

## Назначение

- **Для людей**: единое место с шаблонами и правилами написания PRD.
- **Для Cursor (AI)**: контекст по запросу — при создании или доработке фичи используйте файлы из этого репозитория.

## Структура

```
cursor-prd-project/
├── README.md           # Этот файл
├── Create-PRD.md       # Шаблон и инструкция: как создать новый PRD
├── docs/
│   └── prds/          # Готовые PRD: docs/prds/PRD-[feature-name].md
└── .gitignore
```

## Как использовать

1. **Создать новый PRD**  
   Откройте `Create-PRD.md`, скопируйте шаблон или попросите Cursor:  
   *«Создай PRD по Create-PRD.md для фичи [название]»*  
   Сохраняйте результат в `docs/prds/PRD-[feature-name].md`.

2. **Доработать существующий PRD**  
   Укажите Cursor файл PRD и задачу, например:  
   *«По docs/prds/PRD-xxx.md добавь пользовательскую историю про экспорт»*.

3. **Залить на GitHub**  
   Клонируйте репозиторий, добавьте файлы, сделайте коммит и push (см. раздел ниже).

## Требования

- Редактор с поддержкой Markdown (VS Code, Cursor и т.п.).
- Для работы с Cursor — при необходимости подключайте этот репозиторий или папку в контекст (@ файлы/папки).
